# Rect2f

A rectangle with floating-point components.

## Constructor

```ruby
Rect2f.new
Rect2f.new(x, y, width, height)
Rect2f.new(x: 0.0, y: 0.0, width: 0.0, height: 0.0)
```

- `x` — `Float` — X position
- `y` — `Float` — Y position
- `width` — `Float` — width
- `height` — `Float` — height

## Properties

| Property | Type | Access |
|---|---|---|
| `x` / `x=` | `Float` | read-write |
| `y` / `y=` | `Float` | read-write |
| `width` / `width=` | `Float` | read-write |
| `height` / `height=` | `Float` | read-write |
